
# 🚴‍♂️ Bike Mileage Dashboard (BMD)

A **mobile-first web application** to track bike mileage, earnings, fuel usage, and expenses — with smart insights, goal tracking, and analytics.

Built with a **premium gradient UI + glassmorphism design**, BMD helps riders understand their daily performance and manage savings efficiently.

---

## ✨ Features

### 🏠 Home Dashboard
- Total Earnings 💰
- Total Distance 📍
- Current Mileage ⛽
- Avg ₹/km calculation
- Savings (auto %)
- 🔥 Streak tracking
- 📈 Profit trend indicator
- 🎯 Goal intelligence (daily target + projection)
- 🧠 Smart insights

---

### ⛽ Mileage Tracker
- Petrol entry (KM, litres, cost)
- Real mileage calculation  
- (Current KM - Previous KM) / Litres
- Petrol history table
- Mileage trend chart 📊

---

### 💰 Expense Management
- Add food & other expenses
- Monthly goal progress bar
- Savings tracking
- Expense breakdown (pie chart)

---

### 📊 Insights Page
- Weekly / Monthly analytics
- Earnings trends
- Mileage trends
- Smart recommendations

---

## 🎨 UI Highlights

- 🌈 Gradient dark theme
- 🧊 Glassmorphism cards
- 📱 Mobile app-style layout
- 🔻 Bottom navigation bar
- 📊 Smooth animated charts (Chart.js)

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript
- **Charts:** Chart.js
- **Storage:** LocalStorage (Firebase-ready)
- **Design:** Glassmorphism + Gradient UI
